# deep-thoughts  

## Description
This web application is a social media platform that allows users to post their deepest, innermost thoughts publically. Information is displayed to users in the form of a feed that stays up to date with the most recent thoughts at the top. ## Installation/ Requirements
In order to run this application locally, Node.Js and MySQL have to be installed. To initiate the app, run the `npm run start` command and navigate to `localhost:3001` in your web browser.
